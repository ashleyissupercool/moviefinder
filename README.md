## Dataset

The full dataset (~583MB, ~1M movies & TV shows) is not included in this repository due to GitHub file size limits.

Download the dataset here:
https://drive.google.com/file/d/1ZMN8csRGX2FPS2hkWh7GKfsvcTdDNQCz/view?usp=sharing

After downloading:
1. Unzip the file
2. Place `TMDB_movie_dataset_v11.csv` inside the `data/` folder
3. Run the app normally
